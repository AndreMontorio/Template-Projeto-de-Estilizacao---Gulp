Sobre o Framework
Este é um framework front-end construído para as aplicações da Consinco.
O objetivo é disponibilizar para os desenvolvedores os componentes utilizados na criação de aplicações, bem como, os itens visuais necessários para garantir o padrão do desenvolvimento, como ícones, logos da empresa, fontes e cores.

Vale lembrar que nem todos os componentes são servidos a partir do mesmo local, sendo assim temos o auxilio de frameworks de terceiros para a agilidade no desenvolvimento, sendo eles:
  • - Kendo UI for Vue
  • - Utilitarios para estilizar elementos rapidamente sem usar nenhum código css

Para novas aplicações, o desenvolvedor deve seguir atentamente os modelos aqui expostos. Para auxílio ao desenvolvimento, o site já exibe o resultado final do componente, com o seu estilo aplicado, e a forma de criação.

A Consinco tem como política o uso do navegador Google Chrome e todos os componentes foram desenvolvidos e homologados para esse navegador.
